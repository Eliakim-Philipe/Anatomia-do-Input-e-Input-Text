Anatomia do Input e Input Text

    Observações:

  • Dentre os diferentes tipos de "Input" acima está uma demonstração do modelo em que inserimos textos.

  • Com o termo input type="text" é possível criar um box capaz de receber marcações de texto

  • Adicionando input type="text" name="nome da lista a ser criada" é possível filtrar os dados recebidos em seguida para validações

  • Para que exista um valor padrão já preenchido utilizamos input type="text" name"nome da lista a ser criada" value="valor que já aparece preenchido ex:"

  • Para deixar orientações dentro do input podemos usar input type="text" name="lista-de-email" placeholder="digite seu email ex:"
